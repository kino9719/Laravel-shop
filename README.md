# 簡易電商系統

這是一個使用 Laravel 建立的簡易電商系統。

## 環境要求

-   PHP >= 8.2
-   Composer
-   MySQL
-   Nginx

## 安裝步驟

```bash
git clone https://github.com/zeqas/Laravel-shop.git  && cd /Laravel-shop #1 複製到本機
composer install #2 安裝套件
php artisan serve #3 啟動伺服器
php artisan migrate #4 資料庫遷移
```

## 已完成功能

-   產品列表：顯示所有產品的名稱和價格。
-   產品搜尋：根據名稱和價格區間搜尋產品。
-   新增產品：添加新的產品到系統中。
-   編輯產品：修改現有產品的名稱和價格。
-   刪除產品：從系統中移除產品。
